Create a sass folder inside resources and an app.scss file within it:
Import Bootstrap's SCSS in resources/sass/app.scss:
@import 'bootstrap/scss/bootstrap';
Enable Bootstrap's JavaScript components by importing them in resources/js/app.js:
import * as bootstrap from 'bootstrap';
Create a base layout file (e.g., resources/views/layouts/app.blade.php) and include the compiled CSS and JS using the @vite directive:
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Your Laravel App</title>
        @vite(['resources/sass/app.scss', 'resources/js/app.js'])
    </head>
    <body>
        <div id="app">
            <main>
                @yield('content')
            </main>
        </div>
    </body>
    </html>
    In your individual view files (e.g., resources/views/welcome.blade.php), extend this layout and add your Bootstrap-styled content:
    @extends('layouts.app')

    @section('content')
        <div class="container">
            <h1 class="mt-5">Welcome to your Laravel 12 app with Bootstrap!</h1>
            <button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#exampleModal">
                Launch demo modal
            </button>
            <!-- Bootstrap Modal Example -->
            <div class="modal fade" id="exampleModal" tabindex="-1" aria-labelledby="exampleModalLabel" aria-hidden="true">
                <div class="modal-dialog">
                    <div class="modal-content">
                        <div class="modal-header">
                            <h5 class="modal-title" id="exampleModalLabel">Modal title</h5>
                            <button type="button" class="btn-close" data-bs-dismiss="modal" aria-label="Close"></button>
                        </div>
                        <div class="modal-body">
                            ...
                        </div>
                        <div class="modal-footer">
                            <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Close</button>
                            <button type="button" class="btn btn-primary">Save changes</button>
                        </div>
                    </div>
                </div>
            </div>
        </div>
    @endsection
    
Ajustar o vite.config.js
